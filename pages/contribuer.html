Je Contribue !

Le Kit de l'Innovation Métier est un espace de partage d'outils, de bonnes pratiques et d'innovations des agents publics. C'est grâce à l'ensemble de la communauté des innovateurs que les ressources disponibles sur ce site existent. Si vous avez des méthodes, des modèles de documents, des cas d'usages ou autres retours sur des exopériences d'innovations au sein de la fonction publique que vous pouvez partager, c'est ici que vous pouvez le faire. N'hésitez pas, le processus est simple et les bénéfices pour l'ensemble des agents publics peuvent être très importants!

Comment contribuer ?

L'ensemble des ressources disponibles dans le kit sont stockées sur le site Github, via le compte de la communauté de l'innovation métier de l'Etat. C'est donc via Github que vous pouvez contribuer au kit. Voici la marche à suivre :

  1. Je créé un compte sur Github (gratuit) sur la page github.com/join
  2. Je vais sur le site Github Innovmetier et je clique tout en haut à droite sur le bouton "fork" ce qui va créer une copie du site sur mon compte.
  3. Je me rends dans mon compte github dans le répertoire hébergeant les documents appelé "Files"
  4. Je choisis le répertoire de destination des documents en sélectionnant en premier lieu le type  de document (méthode, modèle ou cas d'usage) puis la famille de document correspondante.
  5. J'uploade mon ou mes documents en cliquant sur Upload file puis une fois que j'ai fini, je soumets un "pull request", c'est à dire une demande d'ajouter mes documents au site principal
  6. J'attends la validation du "merge" (généralement dans la journée sauf pendant les vacances ;] )
